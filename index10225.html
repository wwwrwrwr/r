Практическое задание по HTML

1. Создание и просмотр Web-страницы

Создайте файл с расширением .html, откройте его в браузере, чтобы увидеть результат.

Для просмотра исходного кода:

  • В Internet Explorer: Правая кнопка мыши → "Просмотреть код"
  • В Netscape Navigator: Меню "View" → "Page Source"

2. Атрибуты тега <body>

Internet Explorer

Поддерживает: background, bgcolor, text, link, vlink, alink

Netscape Navigator

Поддерживает: background, bgcolor, text, link, vlink, alink

<body bgcolor="#f0f0f0" text="#000000" link="#0000ff" vlink="#800080" alink="#ff0000">

3. Изменение цвета фона и фонового изображения

<body bgcolor="lightblue" background="фон.jpg">

Или с помощью CSS:

body {
  background-color: lightblue;
  background-image: url('фон.jpg');
}

4. Вставка изображения

<img src="my_image.jpg" alt="Мое изображение" width="300" height="200">
